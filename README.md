# Resume Analyzer with AI

An AI-powered web application that analyzes resumes and provides skill-based scoring along with intelligent improvement suggestions.

---

## 🚀 Features

* Upload resume (PDF or Word)
* Select job role
* Get resume score based on skills
* View matched and missing skills
* Get AI-powered suggestions to improve resume

---

## 🛠 Tech Stack

* Python
* Flask
* HTML/CSS
* OpenAI API (for AI suggestions)

---

## 🔥 Key Highlights

* Built a full-stack web application using Flask
* Implemented skill-based resume scoring system
* Integrated OpenAI API for generating smart resume suggestions
* Designed clean and simple UI for user interaction

---

## ⚙️ How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the application:

   ```
   python app.py
   ```

3. Open in browser:

   ```
   http://127.0.0.1:5000
   ```

---

## ⚠️ Note

* AI suggestions require an OpenAI API key and billing setup.
* Without API credits, the app will still work for scoring but AI suggestions may not function.
