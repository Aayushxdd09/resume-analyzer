# Resume Analyzer with AI

An AI-powered web application that analyzes resumes and provides skill-based scoring along with improvement suggestions.

Provides AI-powered suggestions to improve resume quality.

---

## Features

- Upload resume (PDF or Word)
- Select job role
- Get resume score
- See matched and missing skills
- Get AI suggestions

---

## Tech Used

- Python
- Flask
- HTML/CSS
- Ollama (for AI)

  ---


## Key Highlights

- Built a full-stack web app using Flask
- Integrated local LLM (Ollama) for AI suggestions
- Implemented skill-based resume scoring system

## How to run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000

---

## Note

AI suggestions require Ollama running on your system.
