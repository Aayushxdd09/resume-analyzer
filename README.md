# Resume Analyzer with AI

An AI-powered web application that analyzes resumes and provides skill-based scoring along with improvement suggestions.

It also gives AI suggestions to improve your resume.

---

## Features

- Upload resume (PDF or Word)
- Select job role
- Get resume score
- See matched and missing skills
- Get AI suggestions

---

## Tech Used

- Python
- Flask
- HTML/CSS
- Ollama (for AI)

---

## How to run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000

---

## Note

AI suggestions require Ollama running on your system.
