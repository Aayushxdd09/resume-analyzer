# Resume Analyzer with AI

This is a web app where you can upload your resume and get a score based on your skills.

It also gives AI suggestions to improve your resume.

---

## Features

- Upload resume (PDF or Word)
- Select job role
- Get resume score
- See matched and missing skills
- Get AI suggestions

---

## Tech Used

- Python
- Flask
- HTML/CSS
- Ollama (for AI)

---

## How to run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000

---

## Note

AI suggestions require Ollama running on your system.
